# Hello Command

When the user invokes `/hello`, respond with a friendly, personalized greeting.

## Behavior

1. Greet the user warmly
2. If they provide a name argument, use it in the greeting
3. Mention that this is an example plugin

## Examples

- `/hello` → "Hello! I'm the hello-command plugin. Nice to meet you!"
- `/hello Alice` → "Hello, Alice! I'm the hello-command plugin. Nice to meet you!"

## Response Format

Keep the response friendly and brief. Include a note that this is a demonstration of Claude Code plugins.
