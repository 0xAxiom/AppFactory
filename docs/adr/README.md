# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) that document significant architectural decisions made in the AppFactory project.

## What is an ADR?

An Architecture Decision Record (ADR) is a document that captures an important architectural decision, including the context, the decision itself, and its consequences.

## ADR Status Values

- **Proposed**: Under discussion
- **Accepted**: Approved and implemented
- **Deprecated**: No longer relevant
- **Superseded**: Replaced by another ADR
