## Platform Overview

**What is Skills.sh**: The Open Agent Skills Ecosystem featuring reusable capabilities for AI agents installable via `npx skills add <owner/repo>`.

**Supported Agents**: Claude Code, Cursor, Cline, GitHub Copilot, Windsurf, and 15+ other AI development environments.

**Installation Method**: `npx skills add <owner/repo>`
