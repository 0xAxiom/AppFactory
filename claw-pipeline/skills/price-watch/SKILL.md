# Price Watch Skill

## Overview

Monitor token prices and market data.

## Commands

- `price` — Check current token price
- `watch <threshold>` — Set price alert
- `market` — Show market summary

## Requirements

- Token must be launched (token-receipt.json present)
- Internet connectivity for API calls

## Supported Chains

- Solana: via DexScreener / Birdeye API
- Base: via DexScreener / Uniswap subgraph
