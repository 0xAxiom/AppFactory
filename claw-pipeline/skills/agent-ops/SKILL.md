# Agent Operations Skill

## Overview

Core agent management capabilities for the bot.

## Commands

- `status` — Show bot health and uptime
- `agents` — List active sub-agents
- `delegate <agent> <task>` — Delegate task to sub-agent
- `recall` — Recall active sub-agent to primary

## Sub-Agent Management

Sub-agents (scout, builder, watcher) are managed through this skill.
See AGENTS.md for delegation rules.
