React Native - Running on iOS Simulator

Key Steps to Run on iOS Simulator

  1. Navigate to your React Native project directory
  2. Run the command:
    • npm run ios (using npm)
    • yarn ios (using Yarn)

Device Specification Options

  • Default simulator is "iPhone 14"
  • Specify a different device using --simulator flag
  • Example: npm run ios -- --simulator="iPhone SE (3rd generation)"
  • Can also specify iOS version: npm run ios -- --simulator="iPhone 14 Pro (16.0)"
  • Can use device UDID: npm run ios -- --udid="AAAAAAAA-AAAA-AAAA-AAAA-AAAAAAAAAAAA"

Pro Tip

To see available devices, run xcrun simctl list devices in the console.

Expected Outcome

If everything is set up correctly, you should see your new app running in the iOS Simulator shortly.

Expo Integration

  • For Expo projects, use npx expo start --ios
  • Expo CLI automatically detects and launches iOS simulator
  • Can specify simulator with environment variable: EXPO_IOS_SIMULATOR_DEVICE_NAME
