RevenueCat React Native SDK Installation

Requirements

  • React Native version > 0.64
  • iOS deployment target: â‰¥ 13.4
  • Android minimum API: 23 (6.0)

Installation Methods

Auto-linking (Recommended):

npm install --save react-native-purchases
# or
yarn add react-native-purchases

Manual linking:

  • Install package
  • Run: `react-native link react-native-purchases`

Additional Setup Steps

Android:

  • Set `launchMode="standard"` in AndroidManifest.xml
  • Add Android billing permission:
<uses-permission android:name="com.android.vending.BILLING" />

iOS:

  • Enable In-App Purchase capability in iOS project settings

Import SDK

import Purchases from 'react-native-purchases';

Expo Support

  • Refer to separate Expo-specific installation guide
  • No config plugins required for basic setup

Potential Build Issues

  • For R8 dependency conflicts, update project-level `build.gradle` with specific R8 version configuration
