## Design Specifications

Based on Stage 08 brand identity:

### Color Palette
- Primary: Deep blue (#1565C0) - reliability and professionalism
- Accent: Amber/Orange (#FF9800) - anomaly highlighting and CTAs  
- Background: Rich black (#121212) - professional dark theme

### Icon Concept
- Stylized audio waveform with subtle anomaly spike
- Professional color scheme using primary deep blue and accent orange
- Clean, minimal design conveying audio analysis purpose
- Scalable design working at app icon size and full presentations

### Typography
- Clean, modern sans-serif (SF Pro on iOS, Roboto on Android)
- Professional aesthetic avoiding entertainment/gaming visual language
- Focus on data clarity and precision

## Asset Generation Notes

For production deployment, these placeholder assets should be replaced with:

1. **Professional app icon** designed according to platform guidelines
   - iOS: 1024x1024 PNG, no transparency, no rounded corners
   - Android: 1024x1024 PNG with adaptive icon specifications
   
2. **Splash screens** optimized for various device sizes
   - Should match app's dark theme (#121212 background)
   - Include EVP Analyzer Pro branding elements
   
3. **Web assets** for Expo web deployment
   - Favicon and web app manifest icons
   - Progressive web app (PWA) assets if applicable

## Brand Compliance

All assets must follow the brand guidelines from Stage 08:
- Maintain professional aesthetic suitable for paranormal investigators
- Use approved color palette consistently
- Convey scientific reliability and authenticity
- Avoid entertainment or gaming visual elements
- Support accessibility requirements (sufficient contrast)

## File Formats

- **PNG**: For all app icons and splash screens (lossless, supports transparency)
- **SVG**: For scalable vector graphics (if platform supports)
- **ICNS/ICO**: Platform-specific icon formats (generated from PNG masters)
