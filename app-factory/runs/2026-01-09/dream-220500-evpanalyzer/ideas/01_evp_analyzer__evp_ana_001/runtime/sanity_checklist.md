# EVP Analyzer - Sanity Checklist

## Critical (Must Pass)
- [ ] App launches without crash
- [ ] Onboarding completes and persists
- [ ] Recording starts and stops
- [ ] Waveform renders during recording
- [ ] Session saves and loads
- [ ] Playback works
- [ ] Subscription purchase flow works
- [ ] Premium features gate correctly

## High Priority
- [ ] Session limit enforced (3 for free)
- [ ] Tags can be added and saved
- [ ] Export produces valid WAV file

## Medium Priority
- [ ] Settings persist across launches
- [ ] Location tagging works
- [ ] Spectrogram generates correctly

## Low Priority
- [ ] Haptic feedback fires
- [ ] Animations smooth
- [ ] All empty states display correctly
