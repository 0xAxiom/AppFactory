# Contributing to AppFactory

Thank you for your interest in contributing to AppFactory! This guide will help you get started.

For detailed contribution guidelines, please see [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md).

## Quick Start

1. **Fork the repository** and clone your fork
2. **Install dependencies**: `npm install`
3. **Run tests**: `npm test`
4. **Create a feature branch**: `git checkout -b feature/your-feature`
5. **Make your changes** and ensure tests pass
6. **Submit a Pull Request**

## Development Setup

```bash
# Clone the repository
git clone https://github.com/MeltedMindz/AppFactory.git
cd AppFactory

# Install dependencies for all packages
npm run bootstrap

# Run the full test suite
npm run test:all
```

## Types of Contributions

- 🐛 **Bug fixes**
- ✨ **New features**
- 📚 **Documentation improvements**
- 🧪 **Tests**
- 🔧 **Tooling and infrastructure**

## Before Submitting

- [ ] Tests pass (`npm run test:all`)
- [ ] Code is formatted (`npm run format`)
- [ ] Linting passes (`npm run lint`)
- [ ] Security audit passes (`npm run deps:audit`)
