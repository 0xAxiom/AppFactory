# Hello Agent

## Overview

- **Name**: hello-agent
- **Version**: 1.0.0
- **Interface**: HTTP (port 3000)

## Description

A simple greeting agent that demonstrates the Agent Factory pattern. Takes a name and responds with a time-aware personalized greeting.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | / | Agent info |
| GET | /health | Health check |
| POST | /process | Process greeting |

## Request/Response

**POST /process**

Request:
```json
{
  "input": "World"
}
```

Response:
```json
{
  "response": "Good afternoon, World! It's currently 2:30 PM. Welcome to Agent Factory!"
}
```

## Environment Variables

None required. This example works without any API keys.

Optional:
- `PORT` - HTTP server port (default: 3000)

## Running Locally

1. `npm install`
2. `npm start`
3. Test: `curl -X POST http://localhost:3000/process -d '{"input":"World"}'`
