# Hello Agent - Runbook

## Prerequisites

- Node.js 18+
- npm

## Setup

```bash
cd examples/hello-agent
npm install
```

## Running

### Production mode (compiles TypeScript first)
```bash
npm start
```

### Development mode (hot reload)
```bash
npm run dev
```

## Testing

```bash
# Health check
curl http://localhost:3000/health

# Process a greeting
curl -X POST http://localhost:3000/process \
  -H "Content-Type: application/json" \
  -d '{"input": "World"}'
```
