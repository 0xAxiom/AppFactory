# Mini App Concept

## Name
Hello Mini App

## Tagline
Your first Base mini app

## Description
A simple example mini app demonstrating the MiniApp Pipeline. Say hello to the Base ecosystem!

## Target Users
- Developers learning to build mini apps
- Anyone exploring the Base ecosystem
- Users wanting to test mini app functionality

## Core Loop
1. User opens the app
2. User taps "Say Hello" button
3. App displays a random greeting
4. User can tap again for new greetings

## Category
utility

## Tags
- hello
- example
- starter
- base

## Sharing Context
Users share as an example of what mini apps can do. Developers share to demonstrate the pipeline output.

## Onchain Requirements
none

## Additional Notes
This is a demonstration app to show the MiniApp Pipeline output structure. It includes all required components but focuses on simplicity.
