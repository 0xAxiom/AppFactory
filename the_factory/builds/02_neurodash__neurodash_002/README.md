## Getting Started

### Prerequisites
- Node.js 16 or higher
- Expo CLI (`npm install -g @expo/cli`)
- iOS Simulator or Android Emulator (or physical device)

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm start
   ```

3. **Run on device/simulator**
   ```bash
   # iOS
   npm run ios
   
   # Android
   npm run android
   
   # Web (for testing)
   npm run web
   ```

### Building for Production

```bash
# Install EAS CLI
npm install -g eas-cli

# Configure your project
eas build:configure

# Build for iOS
eas build --platform ios

# Build for Android
eas build --platform android
```

## Architecture

### Core Principles
1. **Energy-First**: All functionality adapts to current cognitive capacity
2. **Supportive**: No shame, guilt, or pressure - only gentle guidance
3. **Accessible**: Built for diverse cognitive and motor abilities
4. **Local-First**: Your data stays on your device

### Key Components

#### Energy System
- `EnergyContext`: Manages energy levels and history
- `EnergyIndicator`: Visual energy display component
- `patternAnalysis`: Learns from user patterns

#### Task Management
- `TaskContext`: Handles task state and operations
- `TaskCard`: Flexible task display with energy matching
- `taskRecommendations`: AI-powered task suggestions

#### Focus System
- Adaptive timer lengths based on energy
- Gentle session management
- Break reminders and celebrations

## User Experience

### Onboarding Flow
1. Welcome with value proposition
2. Energy check-in tutorial
3. Adaptive task demonstration
4. Optional account creation (guest-first)

### Core Usage Flow
1. Energy check-in (voice or tap)
2. View energy-appropriate tasks
3. Flexible task interaction
4. Pattern recognition insights

## Privacy & Security

- **Local Storage**: All data stored on device using AsyncStorage
- **No Analytics**: No user behavior tracking
- **No Account Required**: Full functionality as guest user
- **Export Control**: Users can export their data anytime
