## Brand Strategy

## Brand Guidelines

### Logo Usage
- Minimum clear space: 1/4 logo height
- Minimum size: 24px height
- Never stretch, rotate, or modify
- Use on white, light gray, or brand blue only

### Color Rules
| Color | When to Use |
|-------|-------------|
| Blue | Primary actions, brand moments |
| Green | Positive/success states only |
| Orange | Warning/attention states |
| Gray | Inactive/secondary elements |

### Accessibility Requirements
- All text meets 4.5:1 contrast minimum
- System fonts for optimal rendering
- Dynamic Type support up to 200%
- Inclusive, gender-neutral language
